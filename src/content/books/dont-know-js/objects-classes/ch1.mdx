title: "对象基础"
description: "JS 中的一切都是对象。
这是关于 JS 的最普遍但最不正确的“事实”之一。让我们开始揭穿谣言吧。

JS 肯定有对象，但这并不意味着所有值都是对象。尽管如此，对象可以说是该语言中最重要（也是最多样化！）的值类型，因此掌握它们对于您的 JS 之旅至关重要。

对象机制无疑是最灵活、最强大的容器类型——你可以将其他值放入其中；你编写的每个 JS 程序都会以某种方式使用它们。但这并不是对象在本书中占据首位的原因。对象是 JS 三大支柱中的第二个支柱的基础：原型。

为什么原型（以及this本书后面介绍的关键字）对 JS 如此重要，以至于成为其三大支柱之一？除其他外，原型是 JS 的对象系统表达类设计模式的方式，类设计模式是整个编程中最广泛依赖的设计模式之一。

因此，我们的旅程将从对象开始，建立对原型的完整理解，揭开this关键字的神秘面纱，并探索系统class。"
